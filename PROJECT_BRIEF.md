### Typography Direction
- Headings: Bold, clean sans-serif (Inter or Geist)
- Body: Regular weight, high readability
- Accent text: Cyan (`#29ABE2`) for highlighted words within navy headings
  - Example: **"ABOUT** `US`" — "US" in cyan, "ABOUT" in navy

## 15. Tech Stack for This Website

| Layer      | Choice                          | Reason                                              |
|------------|---------------------------------|-----------------------------------------------------|
| Framework  | Next.js 14 (App Router)         | SSR/SSG for SEO, file-based routing for detail pages |
| Styling    | Tailwind CSS                    | Fast, consistent, utility-first                      |
| Language   | TypeScript                      | Type safety for data files and components            |
| Data       | Local TypeScript files in `src/data/` | No CMS needed for launch; easy to upgrade later |
| Deployment | Vercel (recommended)            | Native Next.js hosting, free tier available          |
